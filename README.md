# Multichannel_Biomaterial_Corrosion_Testing_Cell

Biomaterials must undergo strenuous material testing; the biocompatibility, osseointegration, as well as the corrosion and tribocorrosion properties of the material must be fully evaluated when developing novel biomaterials. The design of a multichannel corrosion testing cell allows for four separate biomaterial corrosion experiments to occur simultaneously, while allowing for cell culture experiments to be conducted with the device due to the biocompatible nature of the material it was manufactured in, polysulfone. The cost of manufacturing such a device was previously too expensive for the corrosion cell to become a widely adopted technology, however the new design of the device allows for it to be fabricated using additive manufacturing, significantly reducing the cost of the technology. With the implementation of an additive manufacturing technique to the advanced material testing design of the corrosion cell, the intention is for the device to allow for further research to be conducted in the field of biomaterials to improve the material properties of dental implants, orthopedic implants, and other medical implants to improve patient outcomes.
